#Standalon Usage
## Install
In order to use `hoster` install it globally:
```bash
$ npm install hoster -g
```
##Usage
###Server
Start the Server by typing:
```bash
$ hoster -f /path/to/conf.json -p password
```

####Conf File
```json
{
  "mode": 0,
  "port": 5555,
  "password": "",
  "modules": []
}
```
